KEYBOARD MANUAL

Mga Pangunahing Keys sa Keyboard

Letters (A–Z)

Para mag-type ng salita.

Numbers (0–9)

Para sa pag-type ng bilang.

Spacebar

Panghiwalay ng salita.

Enter

Para kumpirmahin ang ginawa o pumunta sa panibagong linya.

Backspace

Pambura sa mga nilagay na keys.

Delete

Pambura sa kanan o ginagamit sa pagbura ng mga file o folder.

Arrow Keys

Para igalaw ang cursor.

Madalas na gamiting Shortcut Keys

Ctrl + C

Ginagamit upang kopyahin ang napiling teksto, imahe, o file papunta sa pansamantalang imbakan.

Ctrl + X

Ginagamit upang kopyahin at alisin ang napiling aytem mula sa kasalukuyang lokasyon.

Ctrl + V

Ginagamit upang ilagay ang aytem na ginupit o kinopya mula sa pansamantalang imbakan.

Ctrl + Z

Ginagamit upang bawiin ang huling aksyon na ginawa.

Ctrl + Y

Ginagamit upang ulitin o ibalik ang aksyon na binawi.

Ctrl + S

Ginagamit upang itala ang mga pagbabago sa kasalukuyang dokumento o file.

Ctrl + F

Ginagamit upang buksan ang search box at hanapin ang teksto sa dokumento o webpage.

Ctrl + A

Ginagamit upang piliin ang lahat ng nilalaman sa kasalukuyang dokumento.

Alt + F4

Ginagamit upang isara ang kasalukuyang programa o window.

F5

Ginagamit upang i-refresh ang nilalaman ng window o webpage.

Alt + Tab

Ginagamit upang lumipat sa pagitan ng mga bukas na aplikasyon.
