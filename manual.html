Mga Pangunahing Bahagi ng Mouse

Left Click

Para mag-select o mag-open.

Right Click

Para makita ang options/context menu.

Scroll Wheel

Para mag-scroll pataas at pababa.

Paano Gamitin ang Mouse

1. Ilagay ang kamay sa mouse at i-slide ito para igalaw ang pointer sa screen.

2. Left-click para pumili ng file o buksan ang aplikasyon.

3. Double-click para buksan ang folder o programa

4. Right-click para makita ang mga options tulad ng Copy, Paste, at Properties.

5. Gamitin ang scroll wheel para magbasa o mag-navigate sa pahina.
