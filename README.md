# Kompyuter-Manual
User Manual tungkol sa Hardware at Software ng Kompyuter. Makikita dito ang mga Bahagi ng Internal at External na Hardware, Softwares, Mouse, at Keyboard. Ibinabahagi dito kung paano gamiting ang bawat isa at para saan ito ginagamit.


Para saan ito:
Itong website na ito ay para lamang sa Pananaliksik ng aming grupo sa Filipino sa Iba't Ibang Disiplina ngayong 2nd Year College. Ang pananaliksik na ito ay pinamagatang "Kasanayan ng mga Estudyante sa Teknikal na Pagsulat: Pagbuo ng Web-based na Plataporma ng User Manuals ng Software at Hardware sa Wikang Filipino."

Layunin:
Ginawa ang website na ito para ipakita sa mga gagamit ng kompyuter kung papaano nga ba gumamit ng isang kompyuter.
