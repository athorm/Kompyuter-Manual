KOMPYUTER MANUAL

Ano nga ba ang Kompyuter?

Ang kompyuter ay isang elektronikong makina na tumatanggap, nagpoproseso, at nag-iimbak ng datos ayon sa tagubilin ng gumagamit. Ito ay binubuo ng hardware (pisikal na bahagi) at software (mga programa) na nagtutulungan upang gawin ang iba't ibang gawain tulad ng komunikasyon at paglutas ng problema.

Panloob na Bahagi ng Kompyuter

Ang panloob na bahagi ng kompyuter ay tumutukoy sa mga pisikal na bahagi na nasa loob ng computer case. Ang mga ito ay mahalaga para gumana ang sistema.

Motherboard

Dito nakasalalay ang bilis at kakayahan ng kompyuter dahil dito nakakabit ang CPU, RAM, at iba pang expansion cards. Ito ang pinaka-"kalansay" o foundation ng buong sistema ng kompyuter.

Core Processing Unit

Ang CPU ay ang utak ng kompyuter dahil ito ang nagpoproseso ng lahat ng utos, kalkulasyon, at impormasyon. Ang bilis at dami ng "cores" nito ang nagdidikta kung gaano kabilis at kahusay maisasagawa ng kompyuter ang maraming gawain nang sabay-sabay.

Random-access Memory (RAM)

Ang RAM ay ang pansamantalang workspace o mesa ng trabaho ng kompyuter. Dito inilalagay at pinoproseso ang lahat ng impormasyon na ginagamit ng mga bukas na programa. Dahil ito ay volatile, mawawala ang lahat ng impormasyon dito kapag pinatay mo ang kompyuter.

Hard Disk Drive

Ang Hard Drive ay ang permanenteng pisikal na imbakan ng kompyuter, na nagsisilbing pangunahing lagayan para sa lahat ng mahalagang impormasyon. Dito nakaimbak nang matagalan ang inyong Operating System, lahat ng inyong programa (software), at ang inyong personal na files (litrato, dokumento, at video).

Solid-state Drive (SSD)

Ang SSD ay ang permanenteng imbakan ng kompyuter na gumagamit ng flash memory, kaya mas mabilis itong mag-basa at magsulat ng data kaysa sa lumang Hard Drive. Dahil wala itong gumagalaw na bahagi, nagdudulot ito ng mas mabilis na pag-andar ng Operating System at mga programa, at nagiimbak ito ng inyong mga file nang permanente kahit patayin ang kompyuter.

Graphics Card

Ang Graphics Card na tinatawag ding Graphics Processing Unit (GPU), ay isang chip-based na bahagi na ang pangunahing tungkulin ay magproseso ng graphical data o mga impormasyon tungkol sa larawan, video, at iba pang visuals na ipinapakita sa inyong monitor.

Power Supply

Ang Power Supply Unit (PSU) ang bahagi ng kompyuter na nagko-convert ng kuryente mula sa saksakan tungo sa tamang boltahe na kailangan ng lahat ng internal na bahagi. Ito ang pangunahing pinagmumulan ng enerhiya na tinitiyak na ang buong sistema ay gagana nang maayos at ligtas.

Panlabas na Bahagi ng Kompyuter

Ang External Hardware (tinatawag ding Peripherals) ay ang mga pisikal na kagamitan na nakakabit sa labas ng kompyuter. Ang mga ito ay ginagamit upang magpasok ng impormasyon (Input, tulad ng keyboard at mouse) o magpakita ng resulta (Output, tulad ng monitor at printer).

Keyboard

Ang keyboard ay isang input device na binubuo ng mga tecla o keys na ginagamit upang mag-type ng letra, numero, simbolo, at mga utos papasok sa kompyuter, kayaâ€™t napakahalaga nito sa paggawa ng dokumento, paglalagay ng datos, at pag-navigate sa mga programa.

Mouse

Ang mouse ay isang pointing device na ginagamit upang kontrolin ang cursor sa screen, sa pamamagitan ng pag-click, pag-drag, at pag-scroll, na nagpapadali sa paglipat-lipat sa mga aplikasyon at mas mabilis na pakikipag-interaksyon sa kompyuter.

Speakers

Ang Speakers o Ispiker ay mga output device na naglalabas ng tunog o audio mula sa kompyuter. Mahalaga ito para marinig ang musika, dayalogo mula sa mga video, at mga sound notification ng sistema.

Monitor

Ang Monitor o Palarawan ay ang pangunahing output device na nagpapakita ng biswal na impormasyon. Dito nakikita ang interface ng operating system, ang mga programang tumatakbo, at ang lahat ng tekstong inilalagay mo.

Printer

Ang Printer o Imprentadora ay isang output device na ginagamit upang makagawa ng mga hard copy o pisikal na kopya ng dokumento at imahe mula sa kompyuter, karaniwan ay sa papel.

Software

Ang Software ay tumutukoy sa grupo ng mga programa, mga prosidyur, at mga gawain na ginagamit upang utusan ang kompyuter (o ang hardware) kung ano ang dapat nitong gawin.

1. System Software

Ang nagsisilbing tulay sa pagitan ng hardware ng kompyuter at ng mga gumagamit nito.

Operating System (OS)

Ang OS ang pinakapangunahing programa sa kompyuter na nagpapatakbo at nagma-manage sa lahat ng hardware component at nagbibigay ng user interface (ang nakikita mo sa screen) para makipag-ugnayan ka sa makina.

Device Drivers

Ito ay maliliit na programa na nagsasabi sa Operating System kung paano makipag-ugnayan sa isang partikular na bahagi ng hardware (tulad ng Printer o Graphics Card) upang gumana nang tama.

Utility Software

Mga programa na tumutulong sa pagme-maintain at pagpapabuti ng performance ng kompyuter. Halimbawa nito ay ang antivirus at disk cleanup tools.

2. Application Software

Ito ay ang mga programa na ginagamit ng user upang isagawa ang mga partikular na gawain.

Productivity Software

Ito ang mga programa na tumutulong sa user na maging mas epektibo at organisado sa trabaho o pag-aaral, tulad ng Word Processors (MS Word) para sa dokumento, at Spreadsheets (MS Excel) para sa pagkalkula ng datos.

Web Browsers

Mga programa na ginagamit upang makita at makapag-navigate sa World Wide Web (Internet), tulad ng Google Chrome at Microsoft Edge.

Multimedia/Graphics Software

Ginagamit para sa paglikha, pag-e-edit, o paglalaro ng audio, video, at mga larawan, tulad ng Adobe Photoshop at VLC Media Player.

Communication Software

Ginagamit upang makipag-ugnayan sa ibang tao sa internet, kabilang dito ang Email clients (tulad ng Outlook) at mga Video Conferencing Apps (tulad ng Zoom).

Gaming Software

Lahat ng uri ng mga video games na inilalaro at ini-install sa kompyuter.
